##  Vòng lặp
- 
    for(gia_tri_khoi_tao; dieu_kien_tiep_tuc; buoc_nhay)
    {
        code
    }
- 
    while(dieu_kien_tiep_tuc)
    {
        code
    }
- 
    do {
        code
    }while (dieu_kien_tiep_tuc)
- 
    Break: kết thúc
- 
    Continue: Không muốn thực hiện, bỏ qua toàn bộ các lệnh phía dưới
- 
    Goto: Nhảy cốc đến vị trí đó.
Ví dụ:
    ```cpp
    #include <iostream> 
    #include <math.h>

    int main()
    {	
        int n;
        try_again:
        std::cout << "n= ";
        std::cin >> n;
        if(n < 0 || n % 2==0){
            goto try_again;
        }	
        
        std::cout << "Gia tri cua n la" <<n;
        
    }
    ```
