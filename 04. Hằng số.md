## Hằng số

1. Không thể thay đổi giá trị trong hằng số

    Ví dụ:
    ```cpp
    #include <iostream> 

    int main()
    {
        const int main_var = 0;
        std::cout << main_var << std::endl;
        return 0;
    }
    ```
    Ví dụ:
    ```cpp
    #include <iostream> 
    #define PI 3.14
    int main()
    {
        std::cout << PI << std::endl;
        return 0;
    }
    ```
