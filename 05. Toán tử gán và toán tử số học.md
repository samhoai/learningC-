## Toán tử gán

1. Toán tử gán

    Ví dụ:
    ```cpp
    #include <iostream> 

    int main()
    {
        int main_var = 0;
        std::cout << main_var << std::endl;
        return 0;
    }
    ```
2. Toán tử số +-*/
 Ví dụ:
    ```cpp
    #include <iostream> 

    int main()
    {
        int age = 18;
        int next_age = ++age;
        std::cout << next_age << std::endl;
        std::cout << age << std::endl;
        return 0;
    }
    ```
  Result: next_age = 19 , age = 19 
   Ví dụ:
    ```cpp
    #include <iostream> 

    int main()
    {
        int age = 18;
        int next_age = age++;
        std::cout << next_age << std::endl;
        std::cout << age << std::endl;
        return 0;
    }
    ```
  Result: next_age = 18 , age = 19  Gán cho giá trị next_age rồi mới tăng lên 1
